👨‍💻 Développement Logiciel


Dans le cadre de ce projet, j'ai développé une application Angular utilisant l'API Jikan, une interface non officielle de MyAnimeList. Elle permet aux utilisateurs de rechercher des animes, consulter leurs fiches détaillées, afficher des suggestions pertinentes et filtrer les contenus NSFW pour une navigation sécurisée.

L'application intègre également un mini-jeu "Guess the Character", dans lequel l'utilisateur doit deviner le nom de l’un des 100 personnages les plus populaires à partir de son image.
